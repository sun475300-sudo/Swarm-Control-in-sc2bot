## ? 발생한 문제 및 해결 과정

### 1. 병력 생산 문제 (2026-01-15)

#### 문제 상황
봇이 게임 중에 병력(유닛)을 생산하지 못하는 문제가 발생했습니다.

#### 발견된 주요 문제들

**1.1 KeyError: 901 (Invalid UnitTypeId)**
- **원인**: 존재하지 않는 UnitTypeId 사용
- **해결**: 유효한 UnitTypeId만 사용하도록 수정

**1.2 await 누락으로 인한 생산 마비**
- **원인**: `larva.train()`이 coroutine으로만 생성되고 `await`가 호출되지 않음
- **해결**: 전체 생산 루틴의 비동기 구조 재설계

**1.3 Race Condition으로 인한 중복 건설**
- **원인**: 여러 매니저가 동시에 "산란못 없음"으로 판단
- **해결**: Construction Reservation Flag 설계로 중복 건설 방지

---

### 2. 자원 관리 문제

#### 미네랄 누적 문제
- **문제**: 미네랄만 8,000 이상 쌓이고 가스 부족으로 고급 테크 유닛 생산 중단
- **해결**: 미네랄이 일정 수치를 넘으면 저글링 대량 생산하는 비상 플러시 모드 추가

### 3. 확장 문제
