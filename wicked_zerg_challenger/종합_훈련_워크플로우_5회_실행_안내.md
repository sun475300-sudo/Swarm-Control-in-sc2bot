## ? 워크플로우 단계

각 반복마다 다음 단계가 순차적으로 실행됩니다:

### 1. 정밀검사 (완료)
- 이미 완료된 상태로 스킵

### 2. 게임 학습 시작
- `run_with_training.py` 실행
- 실시간 게임 창 표시
- 통계 자동 저장

### 3. 학습 완료 후 로직 검사 및 에러 수정
- `tools/auto_error_fixer.py` 실행
- 자동 에러 수정

### 4. 파일 전체 로직 검사
- `tools/code_quality_improver.py` 실행
- 코드 품질 검사 및 개선

### 5. 훈련 시작
- `run_with_training.py` 재실행
- 추가 훈련 세션

### 6. 리플레이 비교 학습 프로그램 실행 및 데이터 적용
- `local_training/scripts/replay_build_order_learner.py` 실행
- 리플레이에서 빌드오더 학습
- 학습된 데이터 자동 적용

### 7. 리플레이 학습 데이터 비교 분석 및 학습 실행
- `local_training/strategy_audit.py` 실행
- 프로게이머 vs 봇 비교 분석
- 전략 개선점 도출

### 8. 5번 반복
- 위 과정을 총 5회 반복
